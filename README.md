# Exploratory Data Analysis (EDA) - Iris Dataset

## Objective
To perform exploratory data analysis on the Iris dataset to understand patterns and relationships.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
- Iris Dataset

## Steps Performed
- Generated summary statistics using describe()
- Visualized distributions using histograms
- Created pairplot to analyze relationships
- Generated correlation heatmap
- Used boxplot to detect outliers

## Output
- Identified patterns and relationships in the dataset
- Visualized feature distributions and correlations

## Conclusion
EDA helps in understanding the dataset and is an important step before building machine learning models.
